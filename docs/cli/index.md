# `hk`

**Usage**: `hk [FLAGS] <SUBCOMMAND>`

**Version**: 0.1.9

- **Usage**: `hk [FLAGS] <SUBCOMMAND>`

## Global Flags

### `-j --jobs <JOBS>`

Number of jobs to run in parallel

### `-v --verbose...`

Enables verbose output

### `-q --quiet`

Suppresses output

### `--silent`

Suppresses all output
